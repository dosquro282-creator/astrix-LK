## Astrix Client (A)

Клиентская часть Astrix:

- Настольное приложение под Windows на Rust.
- Интерфейс, визуально похожий на Discord (4 колонки: сервера, каналы, чат, участники).
- Подключение к серверу B по HTTPS/WSS.
- E2EE‑шифрование текста на стороне клиента.

### Планируемый стек

- Язык: Rust (стабильная версия).
- UI: Tauri + фронтенд (React/Vue/Svelte) либо Rust‑UI (будет уточнено при реализации).
- Сеть: `reqwest`/`tokio-tungstenite` или соответствующие API Tauri.

Детали сборки и запуска будут добавлены по мере реализации клиента.
