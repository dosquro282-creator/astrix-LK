## Astrix Server (B)

Серверная часть Astrix:

- HTTP API для регистрации, логина, управления серверами и каналами.
- WebSocket‑шлюз для событий (сообщения, изменения статуса и т.п.).
- Хранение данных в Postgres, кэш/сессии в Redis.
- Хранение только зашифрованного контента сообщений (e2ee на клиенте).

### Планируемый стек

- Язык: Go (1.22+).
- HTTP/WS: chi/nhooyr или аналогичный стек.
- БД: Postgres (через `pgx`).

Сборка и запуск через Docker/Docker Compose (см. `docker-compose.yml` в корне).
